SASS(Syntactically Awesome Style Sheets)

  • Vendor pre-fix to work across the different browsers
  • Variables to handle css values efficiently ( $textColor: green; )
  • Nesting ( header{ button{} } )
  • Seperate codes in multiple files and Import ( @import 'file name' )
  • Grouping the property and values with @mixin & @include
  • Inheritance ( @extend 'css selector' )
